## 실행 방법
`ncurses` 라이브러리가 없을 경우. 다음 코드를 터미널에서 실행하세요.
```
sudo apt-get update
sudo apt-get install libncurses5-dev libncursesw5-dev
```
그 후 `makefile`을 ` make snakeGame `을 통해서 컴파일 한 후,
`./snakeGame`으로 실행하세요.
